-  `ordinal()` : 해당 상수가 그 열거 타입에서 몇 번째 위치인지를 반환한다.

이 메소드를 열거 타입 상수 

```java
public enum Ensemble {    
	SOLO, DUET, TRIO, QUARTET, QUINTET,    
	SEXTET, SEPTET, OCTET, NONET, DECTET;    
	public int numberOfMusicians() { return ordinal() + 1; }
}
```
